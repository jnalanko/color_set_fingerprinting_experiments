# Work in progress

This repository contain code that was used to run the experiments in [Construction of distinct k-mer color sets via set fingerprinting](https://www.biorxiv.org/content/10.64898/2026.02.16.706153v1).

This repository is still work in progress, but the general idea is that the scripts directory contains python scripts that construct and print bash commands for the experiment runs. These commands assume that Themisto (commit f25bbfd69, with git tag WABI2026-submission), Bifrost and GGCAT are installed at the locations specified by the scripts. The is no single master script to orchestrate everything. If you want to try to reproduce the experiments but are having trouble, Jarno Alanko (that's me) can help you out at alanko.jarno@gmail.com.
